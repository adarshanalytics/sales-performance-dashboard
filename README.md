# Sales Performance Dashboard

## Project Overview
This Power BI dashboard provides insights into sales performance, profit, orders, and quantity across different regions and categories.

## Objectives
- Monitor sales performance
- Track profit trends
- Analyze customer behavior
- Compare regional performance
- Evaluate category-wise sales

## Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Total Quantity

## Features
- Interactive Region Filter
- Interactive Category Filter
- Top 10 Customers Analysis
- Sales by Category
- Profit by Category
- Sales by Region

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization
